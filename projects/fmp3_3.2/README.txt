
	TOPPERS/FMP3 Kernel（Release 3.2）without TECS
		Toyohashi Open Platform for Embedded Real-Time Systems/
		Flexible MultiProcessor Kernel

TOPPERS/FMP3カーネル（TOPPERS/FMPカーネル Release 3.X．以下，FMP3カーネ
ル）は，TOPPERS/FMPカーネルを拡張・改良したもので，ホモジニアスなマ
ルチプロセッサシステム向けのリアルタイムカーネルです．

TOPPERS/FMP3カーネルは，TOPPERS/ASP3カーネルをベースに，マルチプロセッ
サ向けの拡張を加えています

【最初に読むべきドキュメント】

FMP3カーネルの仕様概要がdoc/fmp_spec.txtに，ユーザーズマニュアルが
doc/user.txtにあります．また，従来のTOPPERSカーネルからFMP3カーネルへ
移行するための方法を記述したマイグレーションガイドがdoc/migration.txt
にあります．FMP3カーネルを使用する場合には，まずはこれらのドキュメント
をお読み下さい．

【ファイルの閲覧にあたって】

FMP3カーネルのドキュメント（プレーンテキストファイル）およびソースファ
イルを読む際には，TABを4に設定してください．

【利用条件】

FMP3カーネルの利用条件は，各ファイルの先頭に表示されているTOPPERSライセ
ンスです．TOPPERSライセンスに関するFAQが，以下のページにあります．

	http://www.toppers.jp/faq/faq_ct12.html

【質問・バグレポート・意見等の送付先】

FMP3カーネルをより良いものにするためのご意見等を歓迎します．FMP3カーネ
ルに関する質問やバグレポート，ご意見等は，TOPPERSプロジェクトの会員は
TOPPERS開発者メーリングリスト（dev@toppers.jp）宛またはFMP3カーネルのバ
グトラッキングシステム（Trac）に，その他の方はTOPPERSユーザーズメーリン
グリスト（users@toppers.jp）宛にお願いします．

これらのメーリングリストに投稿されたメールは，ウェブサイトのメーリング
リストアーカイブから読むことができますので，公開すべきでない情報（個人
情報など）をメール中に含めないように注意をお願いします．

TOPPERSユーザーズメーリングリストへの登録方法については，以下のページに
説明があります．

	http://www.toppers.jp/community.html

【ポーティングにあたって】

FMP3カーネルを，TOPPERSプロジェクトからオープンソースにすることを前提に，
未サポートのターゲットにポーティングされる場合には，あらかじめご相談く
ださると幸いです．
